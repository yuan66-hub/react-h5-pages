## 一.安装依赖

```bash
  yarn install
```

## 二.运行(app/pages)开发环境

```bash
  // app 模式
  yarn run dev:app
  // pages 模式
  yarn run dev:pages
```

## 三.打包(app/pages)测试环境

```bash
  // app 模式
  yarn run build:pre
  // pages 模式
  yarn run build:pages
```

## 四.导出分析报告

```bash
  yarn run build:analy
```
